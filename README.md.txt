🎓 Student Performance Predictor
An AI/ML web app that predicts whether a student will Pass or Fail
based on their marks, attendance, and study hours.

🚀 Live Demo
Run locally using Streamlit

🛠️ Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn (Random Forest)
- Matplotlib & Seaborn
- Streamlit

📊 How It Works
1. Enter student marks, attendance, and study hours
2. ML model predicts Pass or Fail instantly
3. Shows confidence percentage and personalised advice

▶️ How to Run
pip install -r requirements.txt
streamlit run app.py

📁 Project Structure
- app.py              → Streamlit web app
- students.csv        → Training dataset
- student_model.pkl   → Trained ML model
- day1 to day4        → Jupyter notebooks
